# Demo Scripts

This directory contains demonstration scripts for various features of the Nexus Agent system.

## Available Demos

### demo_document_processing.py
Demonstrates document processing capabilities, including loading and parsing documents.

### demo_memory_management.py
Shows how the memory management system works, including session and context management.

### demo_rag.py
Demonstrates Retrieval-Augmented Generation (RAG) functionality with vector stores and document retrieval.

### demo_script.py
Basic demonstration script showing core agent functionality.

### demo_tool_calling.py
Shows how the agent can call and use various tools to accomplish tasks.

## Usage

Run any demo script with:
```bash
python demos/demo_name.py
```

Make sure you have the required dependencies installed and the environment configured properly before running the demos.
