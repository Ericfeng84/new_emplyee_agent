# IT 支持文档

## 概述

本文档提供公司IT基础设施、软件使用和技术支持的详细信息。如有IT相关问题，请联系IT支持团队：it-support@company.com 或分机 1003。

## 网络连接

### 公司Wi-Fi

#### Wi-Fi信息
- **网络名称（SSID）**：Company-Guest（访客） / Company-Staff（员工）
- **密码**：请向IT部门获取
- **加密方式**：WPA2-Enterprise（员工） / WPA2-Personal（访客）

#### 连接步骤
1. 打开设备的Wi-Fi设置
2. 选择相应的网络名称
3. 输入密码
4. 如需认证，使用公司邮箱和密码登录
5. 连接成功后，访问公司内网资源

### VPN配置

#### VPN访问需求
- 远程办公时访问公司内网
- 在外网访问公司资源
- 连接公司内部系统

#### VPN软件下载
1. 访问IT支持门户：https://it.company.com
2. 登录后下载VPN客户端
3. 根据操作系统选择对应版本
4. 安装VPN客户端

#### VPN配置步骤
1. 打开VPN客户端
2. 输入服务器地址：vpn.company.com
3. 输入公司邮箱和密码
4. 选择"连接"
5. 首次连接需安装证书，按提示操作
6. 连接成功后，可访问公司内网

#### VPN常见问题
- **连接失败**：检查网络连接，确认密码正确
- **速度慢**：尝试切换服务器节点
- **自动断开**：检查网络稳定性，联系IT支持

## 账户管理

### 公司邮箱

#### 邮箱访问
- **Web邮箱**：https://mail.company.com
- **客户端配置**：
  - IMAP服务器：mail.company.com
  - SMTP服务器：smtp.company.com
  - 端口：IMAP 993, SMTP 587
  - 加密：SSL/TLS

#### 邮箱密码
- 首次登录需修改默认密码
- 密码要求：至少8位，包含大小写字母、数字和特殊字符
- 密码每90天需更新一次

### 系统账户

#### 单点登录（SSO）
- 使用公司邮箱和密码登录所有公司系统
- 支持的系统：HR系统、OA系统、项目管理工具等
- 忘记密码：使用"忘记密码"功能重置

### 密码重置

#### 自助重置
1. 访问 https://account.company.com
2. 点击"忘记密码"
3. 输入公司邮箱
4. 按邮件提示重置密码

#### 联系IT重置
- 如自助重置失败，联系IT支持
- 提供员工工号和邮箱
- IT支持会在1个工作日内处理

## 硬件设备

### 笔记本电脑

#### 新设备申请
1. 向直属经理提交设备申请
2. 说明设备用途和配置需求
3. 经理批准后，IT部门采购
4. 设备到货后，IT部门配置并发放

#### 设备配置
- 操作系统：Windows 11 / macOS
- 办公软件：Microsoft Office 365
- 安全软件：公司标准杀毒软件

#### 设备维护

- 安装安全补丁
- 备份重要数据到公司云盘

- 公司移动设备需安装MDM客户端
- IT部门可远程管理设备

## 数据安全

### 数据备份

#### 自动备份
- 公司云盘自动同步重要文件
- 每日自动备份到公司服务器
- 保留最近30天的备份

#### 手动备份
- 重要项目完成后手动备份
- 使用公司云盘或外部硬盘
- 备份文件命名规范：项目名_日期

### 数据安全规范

#### 敏感数据
- 客户信息：加密存储
- 财务数据：访问权限控制
- 员工信息：仅HR可访问
- 商业机密：禁止外传

#### 数据传输
- 使用公司加密邮件发送敏感信息
- 禁止使用个人邮箱传输工作文件
- 大文件使用公司云盘共享
- 禁止使用公共网盘

### 病毒防护

#### 杀毒软件
- 所有公司设备必须安装杀毒软件
- 定期更新病毒库
- 每周全盘扫描一次

#### 防钓鱼
- 不打开不明邮件附件
- 不点击可疑链接
- 发现钓鱼邮件立即报告IT
